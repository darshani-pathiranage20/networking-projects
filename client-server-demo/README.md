# Client–Server HTTP Demo

## Description
This project demonstrates a basic client–server model using HTTP.

## How It Works
- A Node.js server listens on port 8080
- A client (browser or curl) sends an HTTP request
- The server responds with a text message

## Concepts Covered
- Client–Server architecture
- HTTP request/response
- Ports
- Server vs client

## Tools Used
- Node.js
- HTTP protocol
